# Art ideas bank backend

Для запуска нужно:
- заполнить .env переменными `MAIN_POSTGRES_USER`, `MAIN_POSTGRES_PASSWORD`, 
`MAIN_POSTGRES_DB`, `GARAGE_RPC_SECRET`, `GARAGE_ADMIN_TOKEN`, 
`GARAGE_METRICS_TOKEN`, `S3_ACCESS_KEY`, `S3_SECRET_KEY`, `S3_BUCKET`, `JWT_SECRET_KEY`
- прописать: ```make run-and-attach```
